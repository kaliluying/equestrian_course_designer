# 导出功能用户指南

## 概述

马术障碍赛路线设计工具提供强大的导出功能，支持将您的设计导出为多种格式，包括高质量图片、PDF文档和数据文件。本指南将帮助您充分利用这些导出功能。

## 支持的导出格式

### 1. PNG图片导出
- **用途**: 适用于网页展示、社交媒体分享、演示文稿
- **特点**: 支持透明背景、高分辨率、无损压缩
- **推荐场景**: 在线分享、网站展示、数字媒体

### 2. PDF文档导出
- **用途**: 适用于打印、文档存档、正式报告
- **特点**: 矢量质量、多种纸张尺寸、专业排版
- **推荐场景**: 比赛文档、教学材料、存档备份

### 3. JSON数据导出
- **用途**: 适用于数据备份、系统集成、批量处理
- **特点**: 完整数据保存、可编程处理、跨平台兼容
- **推荐场景**: 数据备份、系统迁移、自动化处理

## 如何导出设计

### 步骤1: 准备导出
1. 确保您的设计已完成
2. 检查所有障碍物位置是否正确
3. 验证路线路径（如果有）
4. 确保已登录账户（导出功能需要登录）

### 步骤2: 选择导出格式
1. 在工具栏中点击"导出设计"按钮
2. 从下拉菜单中选择所需格式：
   - **导出PNG** - 图片格式
   - **导出PDF** - 文档格式
   - **导出为JSON数据** - 数据格式

### 步骤3: 配置导出选项
根据选择的格式，系统会显示相应的配置选项对话框。

## 导出选项详解

### PNG导出选项

#### 缩放倍数
- **1x**: 标准分辨率，文件较小
- **2x**: 高分辨率，适合高清显示
- **3x**: 超高分辨率，适合打印
- **4x-5x**: 极高分辨率，适合大幅打印

**建议**:
- 网页使用: 1x-2x
- 演示文稿: 2x-3x
- 打印用途: 3x-5x

#### 背景颜色
- **白色**: 标准白色背景，适合大多数用途
- **透明**: 无背景，便于后期处理
- **黑色**: 深色背景，适合特殊设计需求

#### 图像质量
- **0.1-0.5**: 低质量，文件小，适合预览
- **0.6-0.8**: 中等质量，平衡质量和大小
- **0.9-1.0**: 高质量，文件大，适合最终输出

#### 水印选项
- 勾选"包含水印"会在导出的图片上添加版权标识

### PDF导出选项

#### 纸张大小
- **A3**: 297×420mm，适合大型设计
- **A4**: 210×297mm，标准文档尺寸
- **A5**: 148×210mm，小册子尺寸
- **Letter**: 216×279mm，北美标准

#### 方向
- **自动选择**: 根据设计尺寸自动选择最佳方向
- **横向**: 强制横向布局
- **纵向**: 强制纵向布局

#### 页边距
可以分别设置上、右、下、左四个方向的页边距（单位：毫米）
- **建议值**: 10-20mm，确保打印时不被裁切

#### 图像质量
- **0.5-0.7**: 较低质量，文件小
- **0.8-0.9**: 标准质量，推荐设置
- **0.95-1.0**: 最高质量，文件大

#### 附加选项
- **包含页脚**: 在页面底部添加生成信息
- **包含元数据**: 在PDF中嵌入设计信息

### JSON导出选项

#### 基本选项
- **包含视口信息**: 保存当前视图设置
- **包含元数据**: 保存创建时间、作者等信息
- **格式化输出**: 使JSON文件易于阅读
- **压缩输出**: 减小文件大小

#### 高级选项
- **缩进大小**: 格式化时的缩进空格数（1-8）
- **选择性包含**: 可以选择只导出特定数据

## 导出过程

### 进度监控
导出过程中，系统会显示详细的进度信息：

1. **初始化阶段**: 准备导出环境
2. **画布准备**: 处理设计元素
3. **SVG处理**: 优化矢量图形
4. **渲染阶段**: 生成最终图像
5. **质量验证**: 检查导出质量
6. **文件生成**: 创建最终文件
7. **完成处理**: 清理临时资源

### 质量检查
系统会自动检查导出质量，包括：
- 路径完整性验证
- 元素渲染准确性
- 整体视觉质量评估

如果发现质量问题，系统会：
- 显示警告信息
- 提供改进建议
- 自动尝试修复（如果可能）

## 常见问题解决

### 导出失败
**可能原因**:
- 网络连接问题
- 浏览器内存不足
- 设计过于复杂

**解决方案**:
1. 检查网络连接
2. 关闭其他浏览器标签页
3. 降低导出分辨率
4. 简化设计复杂度

### 导出质量差
**可能原因**:
- 导出分辨率过低
- 原始设计质量问题
- 浏览器兼容性问题

**解决方案**:
1. 增加缩放倍数
2. 检查原始设计
3. 尝试不同浏览器
4. 使用PNG格式替代PDF

### 文件过大
**可能原因**:
- 分辨率设置过高
- 质量设置过高
- 设计包含大量元素

**解决方案**:
1. 降低缩放倍数
2. 调整质量设置
3. 使用压缩选项
4. 分段导出复杂设计

### SVG元素显示异常
**可能原因**:
- 路径定义不完整
- 样式冲突
- 浏览器渲染差异

**解决方案**:
1. 检查路径设置
2. 重新绘制问题路径
3. 尝试不同导出格式
4. 联系技术支持

## 最佳实践

### 1. 导出前检查
- 确保所有元素都在画布范围内
- 检查路径是否完整连接
- 验证障碍物编号是否正确
- 确认设计符合比赛规则

### 2. 格式选择建议
- **在线分享**: PNG (2x, 白色背景)
- **打印用途**: PDF (A4, 横向, 高质量)
- **数据备份**: JSON (包含完整信息)
- **演示文稿**: PNG (3x, 透明背景)

### 3. 质量优化
- 使用适当的分辨率设置
- 选择合适的背景颜色
- 启用质量验证功能
- 定期保存设计进度

### 4. 文件管理
- 使用描述性文件名
- 定期备份重要设计
- 保存多种格式副本
- 记录导出设置参数

## 技术要求

### 浏览器支持
- **推荐**: Chrome 80+, Firefox 75+, Safari 13+, Edge 80+
- **最低**: 支持HTML5 Canvas和现代JavaScript的浏览器

### 系统要求
- **内存**: 建议4GB以上
- **存储**: 确保有足够空间保存导出文件
- **网络**: 稳定的互联网连接

### 性能优化建议
- 关闭不必要的浏览器扩展
- 确保浏览器是最新版本
- 清理浏览器缓存
- 在导出时避免其他重型任务

## 协作功能

### 协作会话中的导出
在协作模式下导出时：
- 系统会确保导出最新的协作状态
- 其他协作者会收到导出通知
- 导出文件会包含协作者信息

### 权限管理
- 只有登录用户可以导出
- 协作者都有导出权限
- 导出记录会保存在用户账户中

## 故障排除联系方式

如果遇到无法解决的问题：
1. 查看浏览器控制台错误信息
2. 尝试刷新页面重新导出
3. 联系技术支持团队
4. 提供详细的错误描述和截图
